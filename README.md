# Houston Real Estate – Agent Portfolio Website

A responsive real estate portfolio website built for a Houston-area agent, 
featuring property listings, agent profile, and lead capture functionality.

## Tech Stack

- **React 18** – Component-based UI
- **Vite** – Fast development build tool
- **React Router DOM** – Client-side routing
- 
## Features

- 🏠 Property listings page with search/filter
- 👤 Agent profile and bio page
- 📋 Lead capture contact form
- 📱 Fully responsive design

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home |
| `/properties` | Property listings |
| `/schedule` | Scheduling |
| `/contact` | Lead capture form |
| `/about` | About |


## Getting Started

```bash
npm install
npm run dev
```
